# 💬 问题描述

写一个count函数，第一次调用该函数返回1，之后每次调用count，返回的值自动加1

# 示例

count(); // 第一次调用 输出 1
count(); // 第二次调用 输出 2
count(); // 第三次调用 输出 3
1
2
3

# 思路

  1. 通过函数实现局部变量的存储
  2. 需要返回一个函数
  3. 返回的函数赋予count —— 所以可以执行count()
  4. 即执行count绑定的立即执行函数

# 代码实现

var count = (function () {
  var a = 0;
  // console.log(++a);
  // 注意这里不能直接输出结果
  // 应该返回一个函数 —— test才是函数
  return () => {
    console.log(++a);
  };
})();

count(); // 1
count(); // 2
count(); // 3

为什么这里是1呢？是因为count每次调用时，都会执行count函数，也就是会重新走var a = 0, 所以每次调用count()()时，a都是0，所以结果都是1，所以这样也是实现不了上述需求的

所以设置立即执行函数，原因就是为了让a变量受到保护，只有在第一次调用时会执行var a = 0, 之后的调用不会重新走var a = 0这行代码，直接就到了return，所以a的值没有被重新赋值，只会被每次执行count()后进行+1 —— 类似于闭包

var test = (function () {
  var a = 0;
  console.log("执行了一次a = 0");
  return () => {
    console.log("a: ", a);
    console.log(++a);
  };
})();

test();
test();
test();
