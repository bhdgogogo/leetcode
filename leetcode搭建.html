# 实现

1️⃣ 先用vuepress搭一个静态网页，挂载到github上

  1. mkdir
  2. cd
  3. npm init
  4. npm install -d vuepress
  5. 创建docs文件夹 —— docs文件夹就是根目录
  6. 在docs中创建README.md，这时md文件的内容就是首页的内容
  7. 配置package.json，dev和build
  8. 执行yarn dev就可以运行起静态网站了

2️⃣ 挂载到github上

  1. 打包文件生成dist
  2. 建个仓库，建个新分支gh-pages
  3. 把dist中的内容push到仓库的gh-pages分支
  4. 配置github的静态页面路径即可
